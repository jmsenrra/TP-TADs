(p2, a)) \implies (\Pi_1(obtener(p1,a)) \neq \Pi_1(obtener(p2, a)) \land \Pi_2(obtener(p1, a)) = 0 \land \Pi_2(obtener(p2, a)) = 0 \land (\Pi_3(obtener(p1, a)) = 0 \land \Pi_3(obtener(p2, a)) = 0)) $}
